Small uppercase pill for statuses, tags, and meta labels (e.g. "Coming soon", "Insights Partner").

```jsx
<Badge>Coming soon</Badge>
<Badge variant="soft">01 / Hire</Badge>
<Badge variant="solid">New</Badge>
```

`outline` = hairline capsule (default), `soft` = orange wash, `solid` = orange fill. Always uppercase with wide tracking.
