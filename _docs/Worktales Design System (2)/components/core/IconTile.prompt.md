A Font Awesome glyph seated in a rounded orange-wash tile — the universal icon container across pillars, advantages, and menus.

```jsx
<IconTile icon="fa-user-tie" />
<IconTile icon="fa-shield-halved" solid />
```

Default is an orange glyph on a translucent orange wash with an orange hairline. `solid` fills the tile orange with a white glyph and an orange glow — used for guarantees and emphasis. Sizes `sm`/`md`/`lg` = 36/48/56px. Requires Font Awesome 6.4.
