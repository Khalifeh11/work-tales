The uppercase, wide-tracked orange marker that labels every section. Use `plain` above a heading, `pill` for the hero capsule.

```jsx
<Eyebrow>For organizations</Eyebrow>
<Eyebrow variant="pill" pulse>Built in Beirut · Global execution</Eyebrow>
```

`plain` is the bare label (orange, 700, uppercase, 0.12em tracking). `pill` wraps it in a translucent hairline capsule; add `pulse` for the animated orange dot used in the homepage hero.
